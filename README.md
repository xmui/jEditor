# jEditor 1.1.0

jEditor is a minimalist, high-performance image editor optimized for bulk rotation and precise cropping. Inspired by the "Notion x Nothing" aesthetic.

## ✨ Features
- **Instant Bulk Rotation**: Rotate hundreds of photos with zero lag.
- **Precise Cropping**: Mathematically locked bounding boxes that stick to the photo area.
- **PWA Ready**: "Install" it as a desktop app in Chrome or Edge.
- **Memory Optimized**: Handles 1000+ photos without crashing.
- **Deep Dark Mode**: Premium, distraction-free interface.

## 🚀 How to Use
1. **Open**: Drag and drop a folder or specific images into the app.
2. **Review**: Flip through photos using arrow keys or the grid view.
3. **Edit**: Use 'R' to rotate or 'C' to enter precise crop mode.
4. **Save**: Click Save or press Enter. jEditor writes directly back to your local files.

## 📦 Sharing & Hosting
jEditor is a purely client-side tool. You can share it in several ways:
- **Share the link**: If hosted on GitHub Pages or any static server, just share the URL.
- **Standalone File**: Run `npm run build` to generate `standalone.html`. You can send this single file to anyone, and it will work in their browser.
- **Install as App**: Open the link in a Chromium browser and click the "Install" icon in the address bar to add it to your desktop.

## 🛠 Development
To run locally:
1. Clone the repository.
2. Run `npm start` to serve the `app` folder.
3. Open `http://localhost:3000`.
